2. 관련 페이지

	1) movielist.jsp
	
		a. 기본화면
			- 주간 인기리스트
				- select 항목 : 영화 포스터, 리스트 제목, 작성자, 작성일, 좋아요 수, 댓글 수
				- select 조건 : 최근 1주일 이내, 좋아요 많은 순서로 정렬
				[!!] 확인사항 : 조회 수 및 댓글 수 반영여부
			- 전체 리스트
				- select 항목 : 영화 포스터, 리스트 제목, 작성자, 작성일, 좋아요 수, 댓글 수
				- select 조건
					- 기본 : 최신 순 정렬
					- 선택 : 작성일, 좋아요, 조회수, 댓글수
				- 검색 조건 : 글 제목
				
		b. 페이지 이동
			- New List 버튼 : 리스트 작성 페이지로 이동
			- List : 해당 리스트의 내용조회 페이지로 이동
			- 페이징처리 필요
			[!!] 확인사항 : 유저 페이지 이동 여부
			
			
			
	2) listdetail.jsp
	
		a. 기본화면
			- List 상세내용
				- select 항목 : 리스트 제목, 작성자 프로필 사진, 작성자, 조회수, 작성일, 본문 내용, 영화 포스터, 태그, 좋아요/싫어요
			- 댓글
				- select 항목 : 댓글 작성자 프로필 사진, 댓글 작성자, 댓글 작성일, 댓글 내용
				- select 조건 : 작성일 순서로 정렬
				
		b. 페이지 이동
			- 이전 버튼 : 이전 페이지로 이동
			- 유저 정보 : List/댓글 작성자의 상세페이지 이동
			- 영화 포스터 : 해당 영화 상세페이지로 이동
			[!!] 태그활용 방향 논의 필요
			
		c. 기능

					- 수정/삭제

					- 신고 
			- 댓글
				- 작성자
					- 작성
					- 수정/삭제
				- 外
					- 신고
